# Asset Folders

This directory contains all static assets for the Aadhar Civil Consultancy website.

## Structure

- **images/** - Project images, gallery photos, and other visual assets
- **brochures/** - Company brochure PDFs and downloadable documents

## Usage

### Images
Place your project photos and gallery images in the `images` folder. These will be accessible at:
```
/assets/images/your-image.jpg
```

### Brochures
Place your company brochure PDF in the `brochures` folder. The main brochure should be named:
```
Aadhar_Civil_Consultancy_Brochure.pdf
```

This file will be downloaded when users submit the brochure request form.

## File Naming Conventions

- Use descriptive names
- Use underscores or hyphens instead of spaces
- Use lowercase for consistency
- Example: `project_commercial_building_2024.jpg`
